# tasty-bite-data-analysis
Tasty Bite sales data analysis using Excel, Pivot Tables and PPT

## Tools Used
- Microsoft Excel
- Pivot Tables
- PowerPoint

## Project Overview
This project analyzes Tasty Bite sales data to understand product performance,
regional trends, and monthly sales patterns.

## Work Done
- Data cleaning and formatting in Excel
- Created Pivot Tables for analysis
- Built charts and dashboards
- Presented insights using PowerPoint
