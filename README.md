# Tasty Bite Profitability Analysis
Tasty Bite sales data analysis using Excel, Pivot Tables and PPT

## Tools Used
- Microsoft Excel
- Pivot Tables
- PowerPoint (for reporting)

## Project Overview
This project analyzes product pricing and profitability for TastyBites, a multi-city food delivery app.
The goal is to identify high and low-profit products, understand discount impacts, and recommend pricing strategies to maximize profit.

## Work Done
- Data cleaning and formatting in Excel
- Created Pivot Tables for analysis
- Built charts and dashboards
- Presented insights using PowerPoint

- ## Key KPIs
- Revenue per Product
- Profit Margin per Product
- Units Sold
- Discount Impact
- Contribution to Total Profit

- ## Recommendations
- Quick Wins: Promote high-profit products, adjust discounts for low-profit items.
- Long-term Strategies: Implement price optimization, bundling, and focus on profitable categories.
